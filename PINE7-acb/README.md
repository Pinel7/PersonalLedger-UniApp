# 个人记账本 App - 使用说明

## 项目简介

这是一款基于 uni-app + Vue 3 开发的跨平台个人记账应用，支持 iOS、Android、微信小程序等多平台运行。应用采用本地数据存储，保护用户隐私，提供简洁强大的记账功能。

## 核心功能

### 1. 快速记账（首页）
- **本月概览**：顶部显示本月收入、支出、结余
- **类型切换**：支持收入和支出快速切换
- **金额输入**：大字体金额输入，自动格式化
- **分类选择**：15种预设支出分类 + 7种收入分类，带emoji图标
- **备注功能**：可选添加文字备注
- **时间选择**：支持补录历史账单
- **最近记录**：显示最近10条交易记录

### 2. 统计分析（统计页）
- **月份切换**：可查看任意月份的统计数据
- **月度概览**：收入、支出、结余一目了然
- **分类饼图**：Canvas绘制的支出分类占比图
- **分类列表**：详细列出各分类金额和占比
- **趋势分析**：近7天收支柱状图对比

### 3. 预算管理（预算页）
- **总预算设置**：设置月度总支出预算
- **进度展示**：可视化进度条显示预算使用情况
- **超支预警**：80%黄色预警，100%红色警示
- **分类预算**：可为特定分类设置子预算
- **剩余可用**：实时显示剩余预算金额

### 4. 账单搜索（搜索页）
- **关键词搜索**：按备注或分类名称搜索
- **类型筛选**：只看收入或支出
- **分类筛选**：按具体分类过滤
- **日期范围**：选择起止日期查询
- **金额区间**：按金额范围筛选
- **结果详情**：点击可查看和编辑记录

### 5. 账单详情（详情页）
- **详细信息**：查看单条记录的完整信息
- **编辑功能**：修改备注和时间
- **删除功能**：确认后可删除记录

## 技术架构

### 技术栈
- **框架**：uni-app (Vue 3)
- **语言**：JavaScript
- **样式**：SCSS
- **存储**：uni.setStorageSync / uni.getStorageSync (本地存储)
- **图表**：Canvas API (原生绘制)

### 数据模型

#### 交易记录
```javascript
{
  id: String,              // 唯一ID (时间戳)
  type: 'income' | 'expense',  // 类型
  amount: Number,          // 金额
  category: String,        // 分类名称
  categoryId: String,      // 分类ID
  note: String,            // 备注
  time: String,            // 日期字符串
  timestamp: Number        // 时间戳(用于排序)
}
```

#### 预算
```javascript
{
  month: String,           // 月份 "2026-04"
  totalBudget: Number,     // 总预算
  categoryBudgets: {       // 分类预算
    [categoryId]: Number
  }
}
```

## 运行方式

### 使用 HBuilderX（推荐）

1. **下载 HBuilderX**
   - 访问 https://www.dcloud.io/hbuilderx.html
   - 下载并安装 HBuilderX

2. **导入项目**
   - 打开 HBuilderX
   - 文件 -> 导入 -> 从本地目录导入
   - 选择本项目文件夹

3. **运行到浏览器（H5）**
   - 点击工具栏的 "运行" 按钮
   - 选择 "运行到浏览器" -> Chrome/Edge
   - 浏览器会自动打开应用

4. **运行到手机模拟器**
   - 连接手机或启动模拟器
   - 点击 "运行" -> "运行到手机或模拟器"
   - 选择设备即可运行

5. **打包发布**
   - 发行 -> 原生App-云打包/本地打包
   - 或发行 -> 小程序-微信/支付宝等

### 使用命令行（需要配置）

如果项目配置了 CLI 工具，可以使用：
```bash
npm install
npm run dev:h5      # 开发H5版本
npm run build:h5    # 构建H5版本
npm run dev:mp-weixin  # 开发微信小程序
```

## 功能演示流程

### 首次使用
1. 打开应用，进入记账页面
2. 点击支出/收入切换类型
3. 输入金额（如：25.50）
4. 选择分类（如：🍔 餐饮）
5. 可选添加备注（如：午餐）
6. 可选选择日期（默认今天）
7. 点击"确认记账"

### 查看统计
1. 点击底部"统计"标签
2. 查看本月收入、支出、结余
3. 查看支出分类饼图和占比
4. 查看近7天收支趋势

### 设置预算
1. 点击底部"预算"标签
2. 点击"编辑"按钮
3. 输入月度预算（如：3000）
4. 点击"确定"保存
5. 可选添加分类预算

### 搜索账单
1. 点击底部"搜索"标签
2. 输入关键词或设置筛选条件
3. 点击"搜索"或"应用筛选"
4. 查看搜索结果
5. 点击记录可查看详情

## 注意事项

1. **数据安全**：所有数据存储在本地，卸载应用会丢失数据，请定期备份
2. **浏览器兼容**：推荐使用 Chrome、Safari、Edge 等现代浏览器
3. **小程序限制**：小程序版本可能受存储容量限制（通常5-10MB）
4. **性能优化**：建议定期清理旧记录，保持应用流畅
5. **隐私保护**：应用不联网，所有数据仅在本地存储
