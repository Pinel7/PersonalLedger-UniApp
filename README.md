# 个人账本 APP 
基于 uni-app 开发的简洁、无广告、本地存储记账工具

## 🌟 功能介绍
- 收入 / 支出记录
- 分类管理（餐饮、交通、工资、红包等）
- 历史账单查看
- 月度收支统计
- 数据本地存储，不上传云端
- 支持 HBuilderX 直接云打包 APK

## 🛠 技术栈
- 框架：uni-app
- 开发工具：HBuilderX
- 语言：Vue3 + JavaScript
- 存储：本地缓存 Storage
- 打包：Android 云打包

## 📱 如何运行与打包
1. 克隆或下载本项目
2. 使用 HBuilderX 打开项目
3. 运行到浏览器/模拟器预览
4. 发行 → 原生App-云打包 → 生成 APK
